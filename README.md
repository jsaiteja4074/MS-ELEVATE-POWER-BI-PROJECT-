# 🌍 Carbon Footprint Visualization Dashboard

A **Power BI-based data visualization project** that analyzes and presents **carbon emissions across different sectors and countries**.

This project helps users understand **which sectors contribute the most to carbon emissions** using interactive dashboards, making environmental data easy to interpret for **students, researchers, and decision-makers**.

---

# 📊 Project Overview

This dashboard transforms raw carbon emission data into **interactive visual insights**, allowing users to:

* Compare emissions across sectors
* Analyze trends over time
* Identify major contributors to carbon footprint

# ✨ Features

### 📊 Interactive Dashboard

* User-friendly interface built with Power BI
* Easy navigation and filtering

---

### 📈 Sector-wise Analysis

* Compares emissions across sectors like:

  * Power
  * Industry
  * Transport
  * Residential
  * Aviation

---

### 🌍 Country-wise Insights

* Displays carbon emissions across multiple countries
* Helps identify global emission patterns

---

### ⏳ Time-based Analysis

* Covers multiple years of data
* Helps track emission trends

---

### 🎯 KPI Indicators

* Number of countries
* Number of years
* Number of sectors

---

# 🏗️ System Architecture

The project consists of the following components:

---

## 📥 Data Source

* Carbon emission datasets from public sources
* Stored in Excel/CSV format

---

## ⚙️ Data Processing

* Data cleaning and transformation using **Power Query**
* Data modeling and relationships

---

## 📊 Visualization Layer

* Built using **Power BI Desktop**
* Includes:

  * Bar charts
  * Pie charts
  * KPI cards
  * Filters and slicers

---

## 🚀 Deployment

* Published using **Power BI Service**
* Accessible via:

  * Web link
  * Organization workspace

---
